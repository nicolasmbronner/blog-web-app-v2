# blog-web-app-v2
A blog web app demo, created with EJS, CSS and JavaScript, Node.js for the back-end. No database.

## Features

- Read articles
- Create, Edit, Delete your articles
- Light/dark mode toggle
- Realtime feedbacks
- Cross platform
